# Guilhas Carnes E-commerce

E-commerce platform developed for a premium butcher shop based in Brazil.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Features

- Product catalog
- Shopping cart
- Checkout via WhatsApp
- Checkout via WhatsApp
- Responsive design
